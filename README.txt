
Реализация модели хакинга для ролевой игры "Стоимость жизни". Джаббер-бот
обеспечивает всю функциональность для борьбы хакеров: взлом сайтов и аккаунтов
пользователей, борьба друг с другом, получение сведений о взломанных системах
и других хакерах. Джаббер-бот требует для своей работы связи с системой
биллинга, которая содержит сведения о людях, компаниях, денежных переводах и
т.п.

Джаббер-бот требует для запуска библиотеку python-xmpp.
ПО биллинга представляет собой веб-приложение на PHP.
