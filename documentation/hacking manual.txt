=================================
РУКОВОДСТВО ДЛЯ ХАКЕРОВ. ВЕРСИЯ 2
=================================

----------------
КАК ПОДКЛЮЧИТЬСЯ
----------------

1. Запустить любой джаббер-клиент (например, Psi с http://lifecost.tv/web.html)
2. Войти под своим хакерским аккаунтом
3. Зайти в конференцию hackers (Join Groupchat на conference.lifecost.tv)

-----------------------
БАЗОВЫЕ КОМАНДЫ ХАКИНГА
-----------------------

Можно давать как в конференции, так и в личном чате с пользователем System

1. connect target (attack|defend)
	taget - это имя клиента
	attack/defend - режим хакинга

	Режим attack позволяет:
		взламывать систему, отвечая на ее вопросы
		атаковать любых участников взлома
		защищать любых участников взлома
		видеть информацию про убитых хакеров
		в случае взлома системы совершать действия с системой

		при этом: может сработать подписка
		при этом: может сработать сигнализация полиции на адрес police@lifecost.tv
		при этом: при отсутствии правильного ответа на вопрос, система наносит
			  повреждения одному из хакеров
		
	Режим defend позволяет:
		атаковать любых атакующих хакеров
		защищать любых участников взлома
		видеть информацию про убитых хакеров

		при этом: защищающий наносит повреждения хакерам сильнее атакующих

2. subscribe target
	подписаться на соот. клиента - в случае атаки придет сообщение через джаббер
3. unsubscribe target
	снять подписку
4. subscriptions
	показать список подписок
5. status
	показать песональный статус хакера (число хитов для следующего взлома)	

--------------------
КОМАНДЫ АТАКИ/ЗАЩИТЫ
--------------------

Если в конференции есть другие (атакующие) хакеры, можно использовать боевые команды:

1. attack code target
	code - шестизначный цифровой код
	target - имя хакера, которого атакуешь

	если атака проходит успешно, с target'а списывается число хитов, которое зависит от уровня атакующего хакера
	
2. defend reverse-code target
	reverse-code - шестизначный цифровой код атаки, записанный в обратном порядке
	target - имя хакера, которого защищаешь
		 защищать не себя можно	
	
	если удалось провести защиту за небольшое время, атака не проходит

3. stat
	показать информацию о системе и текущих участниках конференции, включая оставшееся число хитов
	команда stat может быть повторно запущена через некоторое время

Если у хакера кончаются хиты, он выкидывается из конференции, а всем оставшимся выводится часть информации про истинную сущность хакера (его тэги).

-------------
ВЗЛОМ СИСТЕМЫ
-------------

Хакеры могут отвечать на вопросы системы
За правильный ответ в отводимое время, с системы снимаются хиты.
Число хитов системы зависит от типа взламываемого аккаунта (физ.лицо, компания, ТНК, Lifecost)

-------------------------------
ЕСЛИ ВЗЛОМ БЫЛ СОВЕРШЕН УСПЕШНО
-------------------------------

1) Из конференции выкидываются все защищающие
2) Начинается обратный отсчет, в течение которого все хакеры могут отправить одну из команд, приведенных ниже
3) Каждая из команд занимает определенное время, они указаны в порядке увеличения времени исполнения:
	- showmoney
		состояние денежного счета
	- smallmoney target
		перевести немного денег (5% или 10$) на счет target
	- healthget
		узнать историю болезни (только для физ.лица)
	- lasttransactions
		узнать историю последних транзакций (2 часа)
	- votes
		узнать историю голосования (только для физ.лица)
	- hacklog
		узнать историю взломов данного аккаунта 
	- emplinfo
		узнать информацию о сотрудниках (только для компании)
	- ratinglog
		узнать историю изменения рейтинга
	- alltransactions
		узнать полную историю транзакций
	- clearlogs
		очистить историю взломов данного аккаунта
	- healthadd msg
		добавить текст msg историю болезни
	- bigmoney target
		перевести много денег (25% или 100$) на счет target
	- deface msg
		вывести сообщение msg от данного аккаунта в блог
4) Начинает исполняться команда того хакера, который ввел ее первым
5) Все результаты выводятся в общую конференцию
6) В это время можно также продолжать атаки и защиты

----------------------------------
ЕСЛИ ВЗЛОМ НЕ БЫЛ СОВЕРШЕН УСПЕШНО
----------------------------------

У всех оставшихся защитников и системы восстанавливаются хиты.

----------------
ПОВТОРНЫЙ ХАКИНГ
----------------

После того, как система была взломана, она не может быть атакована некоторое время.
После того, как хакер вышел или был выброшен из конференции, он некоторое время не может атаковать системы (время восстановления зависит от уровня хакера)

------------
НОВЫЕ ХАКЕРЫ
------------

Любой хакер может рекомендоать любого персонажа на полигоне сделать хакером (условно говоря, научить ремеслу).
Для этого нужно отправить запрос в свободной форме на адрес hackbot@lifecost.tv
Персонаж может стать хакером первого уровня (по решению мастеров)

-----------------------
ПОВЫШЕНИЕ СВОЕГО УРОВНЯ
-----------------------

Хакеры могут со временем повышать свой уровень через совершение удачных взломов.
